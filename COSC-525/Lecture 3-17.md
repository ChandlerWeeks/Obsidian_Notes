# Inductive Bias
- Receptive Field & Pooling are what benefit CNNs, which are based on neuroscience. 
- CNN's have some inductive biases however, these are
	- Locality: Assumption that nearby pixels are related
	- translation equivariance: Patterns can appear anywhere in the image (weight sharing)      

Because  MDC are bayesian, we make the assumption the PDF is gaussian, and the decision boundary is linear. 
Assumption that similar instances are close in proximity in the feature space, and the same for the class label.
