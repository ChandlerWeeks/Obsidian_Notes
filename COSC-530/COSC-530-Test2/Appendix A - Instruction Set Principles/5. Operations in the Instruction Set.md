## Operation Types
**Arithmetic and Logical**: Integer arithmetic + logic
**Data Transfer**: Loads & Stores
**Control**: Branches, jumps, procedure calls, returns, & traps
**System**: Operating system calls, virtual memory
**Floating Point**: floating point arithmetic
**Decimal**: decimal point arithmetic (64 bit floats)
**String**: String move, compares, and search
**Graphics**: Pixel and vertex operations

Arithmetic, data transfer, and control are the most important and frequently executed instructions.
