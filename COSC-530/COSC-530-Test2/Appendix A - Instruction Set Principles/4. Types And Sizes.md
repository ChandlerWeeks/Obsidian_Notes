### 1 Byte Types
char / unsigned chars
### 2 Byte Types
Short / unsigned short
### 4 Byte Types
Int / unsigned int / float 
### 4 or 8 Bytes (depending on word size)
Long int / unsigned long int / pointers
### 8 Bytes
Doubles / long long int / unsigned long long int

! Fun Fact !
You can represent a negative number by taking the unsigned representation, inverting the bits, and adding 1

011 (3) -> 100 + 1 -> 101 (-3)
